# Exercice 21: Lecture de fichier

## Objectif
Apprendre à lire des fichiers avec fgets, fscanf, feof.

## Méthodes de lecture
- **fgets()** : lit une ligne complète (avec \n)
- **fscanf()** : lit avec format (comme scanf)
- **feof()** : teste la fin de fichier
- **fgetc()** : lit un seul caractère

## Programme à exécuter
Lit les fichiers créés dans l'exercice précédent.

## Compilation
```bash
cd 10_fichiers/ex_21_lecture_fichier
make
./ex_21_lecture_fichier
```

## Questions
1. Quelle méthode utiliser pour lire une ligne complète ?
2. Comment lire un fichier CSV ?
3. Que retourne feof() exactement ?
