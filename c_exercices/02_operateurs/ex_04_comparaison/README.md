# Exercice 04: Opérateurs de comparaison et logiques

## Objectif
Maîtriser les opérateurs de comparaison et les opérateurs logiques.

## Notions à retenir
- **== !=** : égalité, différence
- **< > <= >=** : strictement plus petit/grand, inférieur/supérieur ou égal
- **&&** : ET logique (vrai si les deux opérandes sont vrais)
- **||** : OU logique (vrai si au moins un opérande est vrai)
- **!** : NON logique (inverse la valeur)

## Point crucial
En C, "vrai" = 1 et "faux" = 0. Toute valeur non nulle est considérée comme vraie.

## Programme à exécuter
Teste tous les opérateurs avec des exemples concrets.

## Compilation
```bash
cd 02_operateurs/ex_04_comparaison
make
./ex_04_comparaison
```

## Questions
1. Que retourne `5 == 5` ? Et `5 = 5` ?
2. Quelle est la différence entre `=` et `==` ?
3. Que donne `10 && 20` ? Pourquoi ?
